# Kattis-cpp

Dette repoet inneholder mine løsninger (ferdige) og ikke fult så ferdige oppgaver. Oppgavene er skrevet i C++, og hver løsning
inneholder en forklaring, hvor "gode" de er eventuelt om den er riktig eller ikke og en generell beskrivelse. 

## - Struktur

-'cpp/' - Alle oppgavene
	- hver oppgave ligger i egen mappe
	- mappenavn = Kattis-oppgavenavn
	- hver mappe inneholder:
		- 'main.cpp' - selve løsningen
		- 'README.md' - info om oppgaven og løsningen
